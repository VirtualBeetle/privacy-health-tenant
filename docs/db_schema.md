# Database Schema — Health Tenant App

## Engine: PostgreSQL 15
## ORM: GORM (AutoMigrate)
## Database name: `health_tenant`

---

## Table: `users`

Shared table for patients and doctors.

```go
type User struct {
    ID           uuid.UUID `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    Email        string    `gorm:"uniqueIndex;not null"`
    PasswordHash string    `gorm:"not null"`
    Role         string    `gorm:"not null"` // "patient" | "doctor"
    CreatedAt    time.Time
}
```

**Valid roles:** `patient`, `doctor`
**Email is unique** across the entire table (doctors and patients share namespace).

---

## Table: `patient_profiles`

One-to-one with users where role = patient.

```go
type PatientProfile struct {
    ID          uuid.UUID `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    UserID      uuid.UUID `gorm:"type:uuid;not null;uniqueIndex"`
    FullName    string
    DateOfBirth *time.Time
    Gender      string
    BloodType   string
    Phone       string
    Address     string
    CreatedAt   time.Time
    User        User `gorm:"foreignKey:UserID"`
}
```

---

## Table: `medical_records`

Multiple records per patient.

```go
type MedicalRecord struct {
    ID            uuid.UUID  `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    PatientID     uuid.UUID  `gorm:"type:uuid;not null;index"`
    DoctorID      uuid.UUID  `gorm:"type:uuid;not null"`
    Diagnosis     string
    Prescriptions string
    TestResults   string
    RecordDate    *time.Time
    CreatedAt     time.Time
    Patient       PatientProfile `gorm:"foreignKey:PatientID"`
    Doctor        User           `gorm:"foreignKey:DoctorID"`
}
```

---

## Table: `appointments`

Multiple appointments per patient.

```go
type Appointment struct {
    ID          uuid.UUID  `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    PatientID   uuid.UUID  `gorm:"type:uuid;not null;index"`
    DoctorID    uuid.UUID  `gorm:"type:uuid;not null"`
    ScheduledAt *time.Time
    Status      string     // "scheduled" | "completed" | "cancelled"
    Notes       string
    CreatedAt   time.Time
    Patient     PatientProfile `gorm:"foreignKey:PatientID"`
    Doctor      User           `gorm:"foreignKey:DoctorID"`
}
```

---

## Table: `emergency_contacts`

Multiple emergency contacts per patient.

```go
type EmergencyContact struct {
    ID           uuid.UUID `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    PatientID    uuid.UUID `gorm:"type:uuid;not null;index"`
    Name         string
    Relationship string
    Phone        string
    Patient      PatientProfile `gorm:"foreignKey:PatientID"`
}
```

---

## Table: `insurance_details`

One or more insurance records per patient.

```go
type InsuranceDetail struct {
    ID           uuid.UUID  `gorm:"type:uuid;default:gen_random_uuid();primaryKey"`
    PatientID    uuid.UUID  `gorm:"type:uuid;not null;index"`
    ProviderName string
    PolicyNumber string
    CoverageType string
    ExpiryDate   *time.Time
    Patient      PatientProfile `gorm:"foreignKey:PatientID"`
}
```

## Relationships Summary

```
users (1) ──── (1) patient_profiles
users (1) ──── (N) medical_records  [as doctor]
patient_profiles (1) ──── (N) medical_records
patient_profiles (1) ──── (N) appointments
patient_profiles (1) ──── (N) emergency_contacts
patient_profiles (1) ──── (N) insurance_details
```

## AutoMigrate Order (in config/db.go)

Must AutoMigrate in dependency order:
1. `User`
2. `PatientProfile`
3. `MedicalRecord`
4. `Appointment`
5. `EmergencyContact`
6. `InsuranceDetail`

---

## Notes

- All primary keys are UUIDs generated by PostgreSQL (`gen_random_uuid()`).
- GORM `AutoMigrate` handles CREATE TABLE and column additions. Never drop columns automatically.
- `*time.Time` (pointer) is used for nullable date fields like `DateOfBirth`, `RecordDate`, `ScheduledAt`, `ExpiryDate`.
